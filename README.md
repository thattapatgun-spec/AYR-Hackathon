# AYRE-Hackathon Competition Submission

## Project Description

Ayr(= is an AI chatting bot that was created to lessen the effects from mental health issues in modern society. Ayr(= was developed to be as human-like as possible to be a companion that can relieve stress and stay by everyone's side, especially those with mental issues.

## Features

### Detection
Detect mood and preferences from voice, text and biometric data.
- Mood analysis

### Adaptation
Use the detected data to adapt interaction to suit users' mood and psychological status.

### Real Life Information
Ability to interact first like sending reels and videos by collecting data from users' preferences.
- Provide advice on methods or places that suit the user's emotional state at that moment, based on the permissions granted for accessing their location.
- Offers real places to calm your mood using GPS to find nearby relaxing locations such as parks, cafes, or quiet spaces.

### Privacy
Your conversation stays only with you and yourself - no judgement by anyone.

## How to Test the AI

1. Download VS Studio
2. Download Live Server extension from VS Studio by clicking Extension on the left side bar
3. View GitHub codes - can select frontend and backend
4. *Note: We cannot show our API key due to privacy concerns*
5. In VS Studio, open terminal by selecting View on the top bar and choose Terminal
6. In terminal, type the following commands:
   ```bash
   mkdir ai
   cd ai
   mkdir frontend
   mkdir backend
   ```
7. After you copy and paste every code based on the structure on GitHub, make sure to save every file
8. Type in the terminal:
   ```bash
   cd backend
   ```
9. Type in the terminal:
   ```bash
   npm install
   ```
10. Save every file
11. Type in the terminal:
    ```bash
    node server.js
    ```
12. Right click on `index.html` file, then click "Open with Live Server"
13. Enjoy Ayr(=
